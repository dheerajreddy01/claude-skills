# Lifestyle → Tech Interface

> Mapping lifestyle domain requirements to technical implementation

## Domain Skills Covered

- `personal-growth` - Life planning and personal growth
- `side-income` - Side income and passive income

## Requirement → Technology Mapping

| Domain Requirement | Technical Implementation | Software Skills |
|-------------------|-------------------------|-----------------|
| Self-media platform | Blog/YouTube/Podcast | `content-platforms` |
| E-commerce side hustle | E-commerce Platform | `e-commerce` |
| Knowledge products | Course Platform | `content-platforms`, `frontend` |
| Automated workflows | Automation Tools | `automation-scripts` |
| Personal brand website | Portfolio/Landing Page | `frontend` |
| Financial tracking | Spreadsheet/App | `data-analysis` |

## Common Combination Patterns

### Pattern 1: Content Creator

**Focus**: Content creation, self-media operations, audience monetization

```yaml
domain_skills:
  - side-income (deep)
  - personal-growth (basic)

software_skills:
  - content-platforms (required)
  - automation-scripts (recommended)
```

**Use Case**: Running a YouTube channel, blog, or podcast

### Pattern 2: E-commerce Side Hustle

**Focus**: E-commerce entrepreneurship, product sales, supply chain management

```yaml
domain_skills:
  - side-income (deep)

software_skills:
  - e-commerce (required)
  - automation-scripts (recommended)
```

**Use Case**: Marketplace selling, custom-built stores, dropshipping

### Pattern 3: Knowledge Product Creator

**Focus**: Online courses, ebooks, consulting services

```yaml
domain_skills:
  - side-income (deep)
  - personal-growth (basic)

software_skills:
  - content-platforms (required)
  - frontend (recommended)
```

**Use Case**: Online courses, membership subscriptions, ebooks

## Technology Stack Recommendations

### Content Platforms

| Use Case | Recommended Stack |
|----------|------------------|
| Blog | Ghost / WordPress / Medium |
| Courses | Teachable / Gumroad / custom build |
| Membership | Patreon / Substack |

### E-commerce

| Use Case | Recommended Stack |
|----------|------------------|
| Platform | Shopify / marketplace apps |
| Custom build | WooCommerce / Next.js Commerce |
| Digital products | Gumroad / Lemonsqueezy |

### Automation

| Use Case | Recommended Stack |
|----------|------------------|
| Workflows | Zapier / Make |
| Scheduling | Buffer / Hootsuite |
| Automation | n8n / Custom Scripts |

## Anti-Patterns to Avoid

| Anti-Pattern | Problem | Better Approach |
|--------------|---------|-----------------|
| Over-optimizing tech too early | Wastes time | Validate product-market fit first |
| No data tracking | Can't optimize | Set up tracking from Day 1 |
| Over-reliance on one platform | Concentrated risk | Build your own owned channels |
