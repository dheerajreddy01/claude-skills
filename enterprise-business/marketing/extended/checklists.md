# Marketing Checklists

> Complete checklists for SEO, A/B testing, advertising, and more

## SEO Optimization Checklist

### On-Page SEO

#### Title Tag
- [ ] Includes primary keyword
- [ ] Length 50-60 characters
- [ ] Compelling for clicks
- [ ] Brand name at the end

#### Meta Description
- [ ] Includes keyword
- [ ] Length 150-160 characters
- [ ] Includes a call to action
- [ ] Unique and engaging

#### Content Optimization
- [ ] Exactly one H1, containing the keyword
- [ ] Clear H2-H6 structure
- [ ] Keyword density 1-2%
- [ ] 3-5 internal links
- [ ] External links to authoritative sources
- [ ] Images have alt tags
- [ ] Content length > 1500 words (for in-depth articles)

#### Technical
- [ ] Page load time < 3 seconds
- [ ] Mobile-friendly
- [ ] HTTPS secure connection
- [ ] Structured data (Schema)
- [ ] No broken links
- [ ] XML Sitemap submitted
- [ ] Robots.txt configured correctly

### Off-Page SEO

- [ ] High-quality backlink strategy
- [ ] Guest posting plan
- [ ] Social sharing
- [ ] Brand mention monitoring
- [ ] Local SEO (if applicable)

## A/B Testing Checklist

### Before the Test

- [ ] Define a clear hypothesis (if X, then Y)
- [ ] Test a single variable
- [ ] Pre-calculate the required sample size
- [ ] Set the test duration (at least 2 weeks)
- [ ] Confirm tracking codes are correct
- [ ] Set the significance level (typically p < 0.05)

### During the Test

- [ ] Don't end the test early
- [ ] Monitor for data anomalies
- [ ] Ensure even traffic distribution
- [ ] Log any confounding factors

### After the Test

- [ ] Significance p < 0.05
- [ ] Effect size is commercially meaningful (>2%)
- [ ] Consider whether a repeat test is needed to confirm
- [ ] Record results and learnings
- [ ] Implement the winning version

### Sample Size Formula

```
Required sample size = (Zα + Zβ)² × 2 × p(1-p) / d²

Zα = 1.96 (95% confidence level)
Zβ = 0.84 (80% statistical power)
p = baseline conversion rate
d = minimum detectable difference
```

**Quick reference**:
- Baseline conversion rate 5%, detecting a 20% relative lift → approx. 8,000 samples/group needed
- Baseline conversion rate 10%, detecting a 20% relative lift → approx. 3,800 samples/group needed

## Email Marketing Checklist

### Before Sending

- [ ] Subject line < 50 characters
- [ ] Preview text configured
- [ ] Sender name correct
- [ ] Recipient list filtered
- [ ] Personalization variables correct
- [ ] All links clickable
- [ ] Images have alt text
- [ ] Unsubscribe link present
- [ ] Displays correctly on mobile
- [ ] Test email sent

### After Sending

- [ ] Open rate normal (>20%)
- [ ] Click-through rate normal (>2%)
- [ ] Unsubscribe rate < 0.5%
- [ ] Hard bounce rate < 1%
- [ ] No spam complaints

### Open Rate Optimization

**Subject line techniques**:
- Spark curiosity: "You missed this..."
- Be specific with numbers: "3 ways to boost conversion by 50%"
- Personalize: "[Name], an exclusive offer for you"
- Create urgency: "24 hours only"

**Avoid**:
- ALL CAPS
- Excessive exclamation points
- Spam trigger words (free, earn money, you've won)
