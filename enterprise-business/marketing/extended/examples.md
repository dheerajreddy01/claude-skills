### ICE Prioritization Scoring

**Formula**: Impact × Confidence × Ease = Score (1-10 each)

| Experiment | Impact | Confidence | Ease | Score |
|------|--------|------------|------|-------|
| Modify CTA button | 7 | 8 | 9 | 504 |
| Add video tutorial | 8 | 6 | 4 | 192 |
| Redesign homepage | 9 | 5 | 2 | 90 |

### Scoring Guide

**Impact**:
- 10: Likely to drive 50%+ growth
- 7-9: Drives 20-50% growth
- 4-6: Drives 5-20% growth
- 1-3: Drives <5% growth

**Confidence**:
- 10: Backed by data
- 7-9: Similar precedent cases exist
- 4-6: Has a theoretical basis
- 1-3: Pure speculation

**Ease**:
- 10: Completed within 1 day
- 7-9: Completed within 1 week
- 4-6: Completed within 1 month
- 1-3: Requires long-term cross-team coordination

## User Psychology Applications

### Cialdini's Six Principles of Persuasion

| Principle | Psychological Mechanism | Marketing Application | Real Examples |
|------|----------|----------|----------|
| **Reciprocity** | Giving first creates an obligation to return the favor | Free value | Free trials, ebooks, tools |
| **Scarcity** | The rarer, the more precious | Limited quantity/time | "Only 3 spots left" "48 hours only" |
| **Authority** | Experts are more credible | Expert endorsement | Expert recommendations, certification badges, cited data |
| **Consistency** | People want to act consistently | Start small, then go big | Free trial → small payment → subscription |
| **Liking** | People we like are more influential | Emotional connection | Brand story, spokespeople, community engagement |
| **Social Proof** | People follow the majority | Herd mentality | Customer testimonials, user counts, reviews |

### Applying Behavioral Economics Biases

#### Anchoring Effect
```
Original price $199 → Sale price $99

The anchor ($199) makes the sale price feel like a better deal
Applications:
- Show the higher-priced plan first
- Display original price alongside discounted price
- Compare against competitors' higher prices
```

#### Loss Aversion
```
❌ "Get a 20% discount"
✅ "Don't miss out on this price"

People feel losses about twice as strongly as gains
Applications:
- "Don't miss out" > "Get"
- Countdown timers
- Limited-quantity reminders
```

#### Paradox of Choice
```
Too many options → decision difficulty → abandonment

Best practices:
- Offer 3 options
- Recommend the middle option
- Clear comparison tables
```

#### Framing Effect
```
❌ "There's a 10% chance of failure"
✅ "There's a 90% success rate"

The same information framed differently produces different feelings
Applications:
- Positive framing
- Emphasize gains over losses
- Use concrete numbers
```

## Conversion Rate Optimization (CRO) Deep Dive

### High-Converting Landing Page Structure

```
┌────────────────────────────────────────────┐
│  1. Above the Fold                          │
│     • Clear value proposition (understood   │
│       within 10 seconds)                    │
│     • Primary CTA button                    │
│     • Social proof (customer logos)         │
├────────────────────────────────────────────┤
│  2. Pain Point Description                  │
│     • 3 core problems                       │
│     • Relatable descriptions                │
├────────────────────────────────────────────┤
│  3. Solution                                │
│     • Product/service introduction          │
│     • 3-5 core features                     │
│     • Use-case showcase                     │
├────────────────────────────────────────────┤
│  4. Social Proof                            │
│     • Customer testimonials (with photos    │
│       for more credibility)                 │
│     • Data proof ("10,000+ users")          │
│     • Media coverage                        │
├────────────────────────────────────────────┤
│  5. FAQ                                     │
│     • Address purchase objections           │
│     • 5-7 common questions                  │
├────────────────────────────────────────────┤
│  6. Final CTA                               │
│     • Restate the value proposition         │
│     • Clear action button                   │
│     • Eliminate risk (money-back guarantee) │
└────────────────────────────────────────────┘
```

### Form Optimization Guide

```
Field count vs. conversion rate:
3 fields: 25%
5 fields: 18%
7 fields: 14%
10 fields: 8%

Each additional field lowers conversion by about 7%
```

**Best practices**:
- Only ask for necessary information
- Break into steps (multi-page forms)
- Use a progress bar
- Show estimated completion time
- Auto-fill (address, email)

### A/B Testing Priority Order

**High impact (test first)**:
1. Main headline/value proposition
2. CTA button (copy, color, placement)
3. Above-the-fold image/video
4. Pricing display format
5. Number of form fields

**Low impact (test later)**:
- Button corner radius
- Font size tweaks
- Image filters
- Background color tweaks

## Marketing Automation Flows

### New User Onboarding Sequence

```
[Sign-up]
    │
    ├─ Immediately: Welcome email (introduction + next steps)
    │
    ├─ D+1: Tutorial email #1 (core features)
    │
    ├─ D+3: Tutorial email #2 (advanced tips)
    │
    ├─ D+7: Success story share
    │
    └─ D+14: Feedback survey
            │
            ├─ Converted → Thank-you email + advanced content
            │
            └─ Not converted → Special offer
                    │
                    ├─ Converted → End
                    │
                    └─ Not converted → Cold customer sequence
```

### Cart Abandonment Recovery Sequence

```
[Cart abandoned]
    │
    ├─ After 1 hour: Reminder email (shows cart contents)
    │   Open rate: 45%, conversion rate: 10%
    │
    ├─ After 24 hours: Social proof (product reviews)
    │   Open rate: 30%, conversion rate: 5%
    │
    └─ After 72 hours: Limited-time offer (10% discount)
        Open rate: 25%, conversion rate: 8%

Estimated total recovery rate: 15-20%
```

### Re-engagement Sequence

```
[Dormant user: inactive for 30 days]
    │
    ├─ D+0: "We miss you" + new feature introduction
    │
    ├─ D+7: Curated content recommendations
    │
    ├─ D+14: Win-back offer
    │
    └─ D+30: Last chance / account deactivation notice
            │
            └─ Still not returned → Move to cold list
```

## Brand Positioning Example

### Brand Positioning Quadrant

```
          High Price
            │
  Luxury    │  Premium
  Brand     │  Brand
            │
────────────┼────────────
            │
  Economy   │  Value
  Brand     │  Brand
            │
          Low Price

    Emotional ←──────→ Rational
```

**Quadrant descriptions**:
- **Luxury Brand**: high price + emotional (LV, Gucci)
- **Premium Brand**: high price + rational (Apple, Tesla)
- **Value Brand**: low price + rational (IKEA, Uniqlo)
- **Economy Brand**: low price + emotional (Coca-Cola, McDonald's)

## Social Engagement Formula

### Engagement Rate Calculation

```
Engagement rate = (Likes + Comments + Shares + Saves) / Reach × 100%
```

### Platform Benchmarks

| Platform | Good | Excellent |
|------|-----|------|
| Facebook | 1-3% | 3-6% |
| Instagram | 3-6% | 6%+ |
| LinkedIn | 2-4% | 4%+ |
| TikTok | 5-8% | 8%+ |

### Tips to Boost Engagement

1. **End with a question**: "What do you think?"
2. **Polls/multiple choice**: makes participation easier
3. **Use emojis**: boosts engagement by 25%
4. **Reply promptly**: respond to every comment within the first hour
5. **Tag relevant accounts**: increases reach

## Choosing a North Star Metric

### By Business Model

| Business Model | Recommended North Star Metric |
|----------|----------------|
| SaaS | MAU or NRR |
| E-commerce | Number of purchasing users or GMV |
| Media | DAU or reading time |
| Marketplace | GMV or number of transactions |
| Social | DAU or number of interactions |

### Traits of a Good North Star Metric

- Reflects user value
- Can be influenced by the team
- Is a leading indicator (not lagging)
- Understandable company-wide
